# Frontend 

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 19.1.6.

> To Access the front is required create a user first at auth api
> Create a loan at backEnd via postman or swagger
> The Begin Page is the login page
> put the email and password
> if is correct, redirect to loan page table with informations from loans
 
## Running the Frontend in src folder

Install dependencies:
```sh
npm install
```  

Start the development server:  
```sh
npm run start
```

Open `http://localhost:4200/` in your browser.
